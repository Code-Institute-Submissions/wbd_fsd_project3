format_quote   About Quotations.Space!

This web site started out as a class project for the Code Institute full-stack Web development course I was taking to refresh my web development skills to use more modern technology instead of the Java Server Pages (JSP) I had been using.

I often use a "quotation database" to learn software development skill on a new system. I've done so for years on everything from a DEC OpenVMS VAX minicomputer as a DCL command line tool, to Visual Basic 6 and VB.NET/Visual Studio desktop apps for Windows, to a Java desktop application (jar) for Windows and MacOS, to a web application written using Java Server Pages and the Eclipse IDE.

So it was natural to do that again for one of the coding projects in this course when we started working with databases. This particular incarnation was written using the GitPod IDE tied to GitHub, and written in Python 3 using the Flask and Materialize CSS frameworks, and using MongoDB (and PyMongo) for the web application and database. Plus a bit o' JavaScript and jQuery.

It's hosted on Heroku at present.
